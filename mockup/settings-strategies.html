注册新策略

注册策略需要：

  1. 在 src/copy_trader/strategies/<name>.py 实现 Strategy Protocol
  2. 在 config/base.yaml::strategies.<name> 中注册 module 路径与默认参数
  3. 提交 PR，CI 会跑 import-linter（确保 strategies 只依赖 core + marketdata）与 pytest
Dashboard 不能直接注册策略——这属于结构性变更，必须走 git PR。本页"注册策略"按钮会跳转到 config/base.yaml 的编辑表单（draft PR 模式）。
