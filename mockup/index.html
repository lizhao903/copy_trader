Dashboard Mockup

copy_trader 静态 UI 草图。共 16 个独立页面，纯静态、离线可用，仅用于驱动 M4 后端实装的 UX 决策。

实装来源：每个页面 head 注释引用对应 spec（openspec/specs/<capability>/spec.md）与 GitHub issue（M4 issue #23–#29 等）。所有数据均为占位假值。
