新建 Runner

Runner 实例 = (venue + account + strategy + params)。新建后状态为 stopped，需要点 Start 才会拉起进程。

后端走 RunnerService.create()。CLI 等价命令：copy-trader registry create --name <n> --venue <v> --account <a> --strategy <s>
用户友好名，账户内唯一；用于 systemd unit / 日志文件命名空间。
从 ExchangeRegistry 注册项中选取。
来自 config/base.yaml::accounts。改 venue 时本下拉自动过滤。
来自 config/base.yaml::strategies。
JSON。仅覆盖策略默认参数中的指定 key。空 = 用策略默认值。
dev 由 COPY_TRADER_ENV 决定，不可在表单中改。

提交后跳转 runners-detail.html，新实例处于 stopped 状态等待 Start。
